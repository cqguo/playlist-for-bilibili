# 自定义播放列表

## 功能介绍

“自定义播放列表”是一款用于 B 站视频页面的浏览器扩展，可以创建多个播放列表、收藏当前视频，并按照指定模式连续播放。

- 支持创建和删除多个播放列表。
- 支持将当前打开的 B 站视频添加到指定播放列表。
- 支持普通视频和分 P 视频，同一 BV 号的不同分 P 会被视为不同项目。
- 支持编辑视频标题、删除列表中的视频。
- 支持展开播放列表并直接打开其中任意视频。
- 支持顺序播放、列表循环、单曲循环和随机播放。
- 视频播放结束后，会根据当前播放模式自动处理下一项。
- 每个播放列表最多可保存 500 个视频。
- 首次使用时会自动创建一个空的“默认播放列表”。
- 播放列表和播放模式保存在浏览器本地，不会自动同步到其他设备。

## 使用手册

### 1. 安装扩展

1. 打开 Chrome 或其他兼容 Chromium 扩展的浏览器。
2. 进入扩展程序管理页面。
3. 开启“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择本项目所在文件夹。
6. 建议将“自定义播放列表”固定到浏览器工具栏。

项目代码更新后，请在扩展程序管理页面点击“重新加载”，并刷新已经打开的 B 站页面。

### 2. 打开功能菜单

点击浏览器工具栏中的扩展图标，可以使用以下三个入口：

- **将本视频添加到播放列表**：把当前 B 站视频收藏到指定播放列表。
- **播放列表管理**：创建、编辑、整理或删除播放列表。
- **选择播放列表**：查看已有播放列表并开始播放。

如果当前标签页不是 B 站视频页面，“将本视频添加到播放列表”按钮将不可用。

### 3. 创建播放列表

1. 点击扩展图标。
2. 选择“播放列表管理”。
3. 在页面上方输入播放列表名称。
4. 点击“新建播放列表”。

播放列表名称不能为空，最长为 80 个字符。新建的播放列表默认为空。

### 4. 添加当前视频

1. 打开需要收藏的 B 站视频页面。
2. 如果是分 P 视频，请先切换到需要收藏的分 P。
3. 点击浏览器工具栏中的扩展图标。
4. 点击“将本视频添加到播放列表”。
5. 在弹出的列表中选择目标播放列表。

添加成功后，扩展会显示目标播放列表中的视频总数。

注意事项：

- 同一“BV 号 + 分 P”不会重复添加。
- 再次添加已存在的视频时，如果页面标题发生变化，扩展会更新保存的标题。
- 单个播放列表达到 500 个视频后，将无法继续添加。

### 5. 管理播放列表

在“播放列表管理”页面中，可以进行以下操作：

- 点击“展开列表”查看播放列表内容。
- 点击视频标题进入编辑状态。
- 按 `Enter` 或点击“✓”保存标题。
- 按 `Esc` 或点击“×”取消修改。
- 点击视频右侧的“删除”将其移出播放列表。
- 点击“删除播放列表”删除整个播放列表及其全部内容。

删除视频或播放列表前，浏览器会显示确认提示。删除操作完成后无法自动恢复。

### 6. 选择并打开播放列表

1. 点击扩展图标。
2. 选择“选择播放列表”。
3. 点击“展开列表”预览视频。
4. 点击某个视频，可以从该视频开始播放。
5. 点击“打开播放列表”，可以从列表第一项开始播放。

空播放列表无法开始播放，需要先添加至少一个视频。

### 7. 使用页面播放列表

从“选择播放列表”页面打开视频后，B 站视频页右侧会显示当前播放列表。

- 当前视频会以高亮状态显示。
- 点击列表中的视频可以立即切换播放项目。
- 标题下方会显示 BV 号和当前播放状态。
- 页面顶部会显示当前位置和视频总数。
- 视频结束后，扩展会按照所选播放模式继续播放。

直接打开普通 B 站视频链接时，不会启用自定义播放列表模式。

### 8. 设置播放模式

可以在视频页面右侧播放列表顶部选择播放模式。

| 播放模式 | 行为 |
| --- | --- |
| 顺序播放 | 按列表顺序播放，最后一个视频结束后停止 |
| 列表循环 | 按列表顺序播放，最后一个视频结束后返回第一项 |
| 单曲循环 | 当前视频结束后重新播放当前项目 |
| 随机播放 | 当前视频结束后随机选择其他项目 |

每个播放列表的播放模式会独立保存，下次打开同一播放列表时继续使用。

### 9. 数据保存说明

播放列表、视频标题和播放模式均保存在当前浏览器的本地扩展存储中。

- 关闭浏览器后数据仍会保留。
- 数据不会自动同步到其他浏览器或设备。
- 当前版本不提供导入、导出或云端备份功能。
- 卸载扩展或清除扩展数据可能导致播放列表丢失。

### 10. 常见问题

#### 无法添加当前视频

请确认当前页面是 `www.bilibili.com/video/` 下的正常视频页面，然后刷新页面并重新打开扩展。

#### 修改代码后页面没有变化

请在扩展程序管理页面重新加载扩展，再刷新 B 站页面。已经打开的页面可能仍在运行旧版内容脚本。

#### 播放列表没有显示

请确认视频是从“选择播放列表”页面打开的，并检查地址栏链接中是否包含 `bili_playlist=1`。

#### 视频没有自动切换

请确认视频已正常播放到结束，并检查当前播放模式。顺序播放模式在最后一个视频结束后会停止。

#### 本地数据读取异常

先重新加载扩展并刷新页面。如果问题仍然存在，请谨慎清除扩展数据；清除后已有播放列表将无法恢复。
